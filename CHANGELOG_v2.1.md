## 🚀 Как обновиться с v2.0 на v2.1

### Вариант A: Полная замена (рекомендуется)

```bash
# 1. Скачайте новый архив
tar -xf soldia_v2_FIXED.tar

# 2. Сделайте backup старой версии
mv soldia_v2_complete soldia_v2_OLD

# 3. Переименуйте новую версию
mv soldia_v2_FIXED soldia_v2_complete

# 4. Перенесите .env файл
cp soldia_v2_OLD/.env soldia_v2_complete/.env

# 5. Готово!
```

### Вариант B: Только обновление index.html

```bash
# 1. Backup старого файла
cp static/index.html static/index_v2.0_backup.html

# 2. Замените файл
cp soldia_v2_FIXED/static/index.html static/index.html

# 3. Очистите кеш браузера (Ctrl+Shift+Delete)

# 4. Перезагрузите страницу (Ctrl+Shift+R)
```
